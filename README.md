# uniapp-template

这是一个 UniApp 项目模板，用于快速启动跨平台应用开发。

## 项目介绍

该模板提供了 UniApp 开发的基础结构和常用配置，帮助开发者快速搭建跨平台移动应用项目。

## 技术栈

- **UniApp**: 基于 Vue.js 的跨平台开发框架
- **Vue.js**: 渐进式 JavaScript 框架

## 项目特性

- 📱 支持多端编译（H5、小程序、App）
- 🛠️ 预配置开发环境
- 📁 标准化的项目结构
- 🔧 集成常用开发工具

## 快速开始

```bash
# 克隆项目
git clone <repository-url>

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev:%PLATFORM%
```

## 构建部署

```bash
# 构建生产版本
pnpm build:%PLATFORM%
```
